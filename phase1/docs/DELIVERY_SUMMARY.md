## 🎯 核心特性

### 1. 灵活的训练策略

| 策略 | 适用场景 | 优势 |
|------|----------|------|
| 预训练 + 冻结编码器 | 数据少（<100） | 最快，防止过拟合 |
| 预训练 + 完整微调 | 数据中等（100-500） | 平衡速度和质量 |
| 从头训练 | 数据多（>500） | 完全自定义 |

### 2. 预训练模型支持

- ✅ **Brain MRI VAE**: 31,740 个 3D 脑部 MRI 训练
  - 分辨率: 160×224×160
  - 最适合 3D 医学图像

- ✅ **Chest X-ray VAE**: 90,000 个 2D 胸部 X 光训练
  - 分辨率: 512×512
  - 适合 2D 医学图像

### 3. 显存优化

- ✅ 混合精度训练（自动）
- ✅ 可配置 batch size 和 patch size
- ✅ 可配置模型通道数
- ✅ 可选数据缓存策略
- ✅ 支持低至 8GB 显存的 GPU

### 4. 完整的测试工具

- ✅ 重建质量评估（MSE, MAE, PSNR）
- ✅ 潜空间平滑性检查（插值）
- ✅ 生成能力测试（随机采样）
- ✅ 3D 可视化（3个正交切面）

## 🚀 快速开始（3步）

### Step 1: 准备数据
```bash
mkdir -p data/vessel_masks
# 将你的 .nrrd 文件复制到这个目录
```

### Step 2: 运行训练
```bash
# Windows
quickstart_vae.bat

# Linux/Mac
bash quickstart_vae.sh
```

### Step 3: 监控和测试
```bash
# 监控训练
tensorboard --logdir ./outputs

# 测试模型（训练完成后）
python test_vae.py --model_path ./outputs/vae_*/best_vae.pth --data_dir ./data/vessel_masks
```
